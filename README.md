\# SmartHealth - Mobile Application



A comprehensive mobile application for medical aid services built with Flutter.



\## 📱 Features



\### User Authentication

\- \*\*Register\*\*: Create a new account with email and password

\- \*\*Login\*\*: Secure login with validation

\- \*\*Profile Management\*\*: Update personal information, phone number, and medical aid details

\- \*\*Logout\*\*: Secure logout functionality



\### Dashboard

\- \*\*Welcome Header\*\*: Personalized greeting with user name

\- \*\*Quick Stats\*\*: View total claims and approved amounts

\- \*\*Quick Actions\*\*: 

&#x20; - Submit new claim

&#x20; - View claim history

&#x20; - Check notifications

&#x20; - Access support



\### Claims Management

\- \*\*View Claims\*\*: See all claims with status indicators

\- \*\*Submit Claims\*\*: Upload receipts and submit new claims

\- \*\*Track Status\*\*: Monitor claim status (Pending, Processing, Approved, Rejected)

\- \*\*Claim History\*\*: Detailed history of all submissions



\### Profile Section

\- \*\*Personal Information\*\*: View and edit name, contact details

\- \*\*Medical Information\*\*: Update medical aid number

\- \*\*Settings\*\*: Manage app preferences

\- \*\*Policy Details\*\*: View membership information



\## 🎨 UI/UX Features



\- \*\*Modern Design\*\*: Clean, professional healthcare-themed interface

\- \*\*Color Scheme\*\*: 

&#x20; - Primary: Trustworthy Blue (#2563EB)

&#x20; - Secondary: Healing Green (#10B981)

&#x20; - Accent: Purple (#8B5CF6)

\- \*\*Smooth Animations\*\*: Page transitions and micro-interactions

\- \*\*Responsive Layout\*\*: Works on various screen sizes

\- \*\*Bottom Navigation\*\*: Easy navigation between sections

\- \*\*Floating Action Button\*\*: Quick claim submission



\## 🛠️ Technologies Used



\- \*\*Framework\*\*: Flutter

\- \*\*Language\*\*: Dart

\- \*\*State Management\*\*: Provider

\- \*\*Local Storage\*\*: SharedPreferences

\- \*\*UI Components\*\*: Material Design 3

\- \*\*Fonts\*\*: Google Fonts (Poppins)

\- \*\*Icons\*\*: Font Awesome \& Material Icons
